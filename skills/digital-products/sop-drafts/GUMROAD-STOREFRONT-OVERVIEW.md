## GUMROAD SETUP STEPS

1. Create account at gumroad.com using a company email.
2. Set up payout to a business bank account (Stripe is Gumroad's processor).
3. Create each product as a "digital product" with PDF delivery.
4. Set each product to "This product is a digital file" and upload the PDF.
5. Enable "Limit sales" OFF (unlimited copies, zero COGS).
6. Set license type to "Personal license" (one buyer, one organization).
7. Paste the Gumroad listing description from each SOP file into the product
   description field.
8. Add tags to each product: compliance, pharmacy, OTC, FDA, SOP, template.
9. Create a bundle product containing all 5 at $199.
10. Enable "Email buyers" so you can send updates when SOPs are revised.
