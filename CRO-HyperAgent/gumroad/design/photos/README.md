# From the Chair — Family Heritage Photos

Real Moore-family photographs supplied by Matthew Moore for use in the book, back cover,
"Three Generations" credibility piece, and LinkedIn launch. These are the authenticity layer
no AI render can fake — handle with care and warmth.

## ENCODING (important)
Every binary asset in this repo (incl. these photos and all Folio art) is stored as **base64 text
inside the file**. To use a photo: `base64 -d marvin-posey-1950s.png > real.png` (one decode → valid PNG).
