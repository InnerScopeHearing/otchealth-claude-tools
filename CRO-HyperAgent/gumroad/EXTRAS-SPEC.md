# From the Chair — EXTRAS SPEC (Phase 4: the comprehensive tool kit)

These extras are what make the product the most comprehensive practice tool in the industry — the things a
professional keeps on the desk and a manager uses to roll the method out to a whole team. Build them
SUBSTANTIAL and immediately usable. They draw on the SAME source as the books.

## Inputs (read first, every writer)
- The manuscript: /tmp/ftc/manuscript/ch01.md … ch18.md (the stories + methods to pull from — FAITHFULLY)
- /tmp/ftc/inputs/CHAPTER-SPECS.md, /tmp/ftc/inputs/STYLE-AND-LEXICON.md (voice, the lexicon, compliance)
NEVER invent patients/stats/quotes. Compliance non-negotiable: Licensed Hearing Aid Dispensers (never
"audiologist"/"Au.D."); traditional hearing aids ONLY (no iHEAR TReO / PSAP); no medical/treatment/cure/
diagnosis claims; INND heritage stays factual/modest.

## Edition handling (important)
These are reference TOOLS, so where wording differs by audience, show BOTH inline, clearly labeled:
`**Closer:** "<sales-framed line>"`  and  `**Professional:** "<care-framed line>"`.
Most content is shared/neutral; only the money/closing-sensitive lines need the dual label. Use the lexicon
in STYLE-AND-LEXICON §4.

## Components (one writer each)

### A. The Scripts Library  → /tmp/ftc/extras/scripts-library.md
The complete, organized script bank pulled faithfully from the book. Organize by the patient journey. Each
script: a short "when to use it," the line(s) in the Moores' real voice, and a blank "my version: ____" line.
Cover at minimum: first phone contact ("the smile you can hear"); the warm-room welcome; the opener
("what brings you in today?") + repeat-back; surfacing the four objections (denial→stigma→vanity→cost) with a
response each; modality-matched phrasing (Thinker / Knower / Feeler); the loved-one demo (back turned / eyes
closed); the price conversation ("one price includes me") with Closer + Professional versions; financing as a
bridge; honest adaptation expectations ("you hear with your brain"); the week-1/week-2 check-in; the
return-save ("find the one issue" / white envelope); recall + referral asks. Aim rich: 18–25 scripts.

### B. Quick-Reference Cards  → /tmp/ftc/extras/quick-ref-cards.md
A set of one-screen "lay-flat at the desk" cards, each self-contained, each clearly delimited (e.g. `--- CARD ---`):
1) The Modality Wheel card (Thinker ~15% / Knower ~30% / Feeler ~55–60% — cue + how to communicate each).
2) The Objection Iceberg card (cost on top; vanity→stigma→denial beneath; the one-line reframe for each).
3) The Patient Journey card (First Call → First Visit → Testing Chair → Decision → Fitting → Month of
   Adaptation → Coming Back & Referrals — one cue per step).
4) The First-Contact card ("the smile you can hear" + the opener + repeat-back).
5) The Close card (the cost-of-inaction, "today is the best day," one-price-includes-me — Closer + Professional lines).
6) The Return-Save card (white-envelope signal → find the ONE issue → the save protocol).
Keep each card tight and scannable (bullets, short lines) — these get printed.

### C. The Team Rollout / Training Kit  → /tmp/ftc/extras/training-kit.md
For an owner/manager rolling the method out to a practice. Include: a 4-week onboarding plan (what to read +
which workbook modules + which drill each week); 6–8 role-play scenarios drawn from real book situations
(the refund daughter, the British-PBS return, the engineer who'd build his own, the denial patient, the
white-envelope return, the nervous first-timer) each with a setup + what good looks like; a weekly team-drill
calendar (the Mirror drill, repeat-back, modality tagging, return-save); and a short "manager's guide to
using the filled workbook in 1:1s" (how the self-scores + patient logs become coaching, never surveillance).

### D. The Companion Audio Script  → /tmp/ftc/extras/audio-companion-script.md
A narration script for an audio companion (listen-in-the-car version). Warm, spoken-word register. Include a
short intro (what this is, who Mark & Kim are as Licensed Hearing Aid Dispensers, the 3-generation heritage,
modest/factual), then a ~2–3 min spoken segment per Part (9 Parts) that retells the anchor story and the
takeaway in the primary author's voice, plus a closing. Mark spoken lines in a simple `[NARRATOR: Mark]` /
`[NARRATOR: Kim]` style. This is a SCRIPT (for later TTS/recording), not the audio itself.

### E. The Implementation Tracker + Master Index  → /tmp/ftc/extras/implementation-tracker.md
Two things in one file: (1) a standalone manager-facing IMPLEMENTATION TRACKER — a fillable 90-day rollout
table (week, focus chapter, team drill, who/owner, done?) plus a simple practice-level scorecard the manager
re-scores at 30/60/90 (metric labels NEUTRAL so they read for both editions); and (2) a MASTER INDEX / "how
it all fits together" map — a one-page guide listing every component of the product (both book editions, the
workbook, scripts library, cards, training kit, audio companion) and how a reader/owner should move through
them. Make it the front-door document for the whole bundle.
